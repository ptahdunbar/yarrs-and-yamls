# Yarrs And Yamls

## Installation

`vagrant plugin install vagrant-yarrs-and-yamls`

## Usage

Replace your Vagrantfile with:
```
Vagrant.configure(2) do |config|
  yarrs_and_yamls(config)
end
```

Create a new file named `Vagrantfile.yml`
